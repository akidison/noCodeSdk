Integration Guide

        

            
1

            

                
Add Pushwoosh SDK

                
Add PushwooshXCFramework and PushwooshXCFramework/PushwooshNoCode via CocoaPods, or link frameworks manually.

            

        

        

            
2

            

                
Add JSON to your project

                
Drag pushwoosh-nocode.json into Xcode. Make sure "Copy items if needed" and your app target are checked.

            

        

        

            
3

            

                
Set your App ID

                
Add Pushwoosh_APPID key to your app's Info.plist with your Pushwoosh Application Code.

            

        

        

            
4

            

                
Enable Push in Xcode

                
Go to your target's Signing & Capabilities, click "+ Capability" and add "Push Notifications".

            

        

        

            
5

            

                
Build & Run

                
That's it! The SDK reads the JSON on launch and executes all actions automatically. Zero code needed.
